 PetAdopt - Pet Adoption Platform

## Purpose
PetAdopt is a full-stack pet adoption platform where users can browse pets, submit adoption requests, and pet owners can manage their listings.

## Features
-User Authentication (Email/Password & Google Login)
-Browse all available pets with search & filter
-Submit adoption requests for pets
-Pet owners can add, edit & delete pet listings
-Approve or reject adoption requests
-Dashboard for managing listings and requests
-JWT Authentication with HTTPOnly cookies
-Fully responsive design
